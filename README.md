# TaskHappy

Smarter to-do lists. Happier you. 🎉

A recurring task manager for households — track personal care, house work, car
maintenance, and any other custom categories you create. Built with React + Vite.

## Features

- Custom categories with your own colors and emoji icons
- Recurring tasks with goal frequencies that adapt based on your actual completion history
- Completion history log with editable entries
- Assign tasks to household members (fixed, alternating, or anyone)
- Cutesy card view or compact sortable table view
- Status filter slider (overdue → due this week → all)
- Bulk import tasks via CSV or Excel (.xlsx) template
- All data stored locally in your browser (localStorage) — no account needed

## Getting Started

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Deploying

This project is ready to deploy on [Vercel](https://vercel.com) — just import
the GitHub repo and it will detect the Vite build automatically. No environment
variables are required.
